### 项目介绍
- 开发环境：windows操作系统
- 开发语言：html+css+jQuery+Ajax
- 开发工具：webstorm+SublimeText
- 责任描述：
  1.使用div+css完成页面布局;
  2.页面有首页、登录和注册、天天特卖、购物车、商品列表、商品详情页等页面;
  3.使用CSS3实现页面的动画效果，提升页面的交互性;
  4.使用ajax实现相同部分(头部和尾部)的复用; 
  5.对用户登录注册使用正则表达式进行验证；
